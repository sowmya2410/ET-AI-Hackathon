Demo Guide

Using the product spec with GCN pipeline

Each downloaded spec contains a pre-written content brief prompt. Paste it directly into the GCN Orchestrator to trigger the full 9-agent agentic simulation.

1
Download a product spec
Click "Download Product Spec" on any product card. Each spec contains product parameters, compliance context, and a ready-made content brief.
2
Copy the GCN Input Prompt
Open the downloaded .txt file and find the "GCN CONTENT BRIEF PROMPT" section. This is the structured input Agent 1 expects.
3
Paste into Orchestrator
Feed the prompt into the GCN pipeline text input. The Orchestrator will trigger Agent 1 through Agent 9, including the human approval gate.
4
Observe agentic simulation
Watch compliance auditing, persona selection, vernacular localisation, channel formatting, and the publisher dispatch — fully simulated end-to-end.
